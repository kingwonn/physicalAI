④ 系统方案设计详解 · 三模块架构

每个装置都可归结为三模块架构:感知装置 → 位姿/动作恢复(SLAM/VIO/ARKit 恢复 或 编码器 实测)→ 数据格式(Zarr/HDF5/RLDS/LeRobot)。设计空间即这三块内部的工程选择,每个选择都在两条轴上下注:标签保真度 与 具身差距。

⑥ 中国生态批判裁决

差异①谁买单:西方分裂为学术 UMI 装置 + 私营遥操作车队;中国多了国家——40+ 数据训练场把采集成本社会化分摊到部委/市级预算、硬件厂(折价供机器人)、廉价劳动(京东居家零工 $3/时 确认;训练中心工时薪未披露,「$5–20/时」未证实)。MIIT+SASAC 实景实训专项(2026-06-09)把每个强制部署场景变成采集点。

差异②硬件厂降本:西方 A 族是开源 BOM(谁都能打印);中国把它产品化——松灵 €16k 目录 SKU,GenRobot/它石/帕西尼卖交钥匙套件。同一条 60–70% 零部件供应链既降本人形、也降本装置本身。中国比较优势 = 廉价、集成、可买的采集硬件走量(军火商楔子)。

差异③异构跨厂数据:国家训练场跑 100+ 机型于同场景 → 设施级跨本体数据(张江 ~10 家 / 亦庄 120+ 机型),西方多单一本体(Figure→Figure)。是否真能训出更好跨本体策略未证——但这是只有国家协调模式能下的注。

诚实裁决:中国的边是成本结构与协调,不是数据质量或可验证性。最严谨的三家(星海图 / AgiBot World / 字节 GR-3)有真论文/数据卡、真竞争。但下一层(千寻「1,000 台/100 城」、逐际「Data Recipe」、京东「1,000 万时」、各训练场「1,000 万条」)多是公司/国家单源,常是计划或目标而非交付的、精选的、可验证数据集。持久护城河不是任何单一数据集——是垂直整合栈(自研机器人 + 自研装置厂 + 国家补贴设施 + 廉价劳动 + 开放切片飞轮),西方无人能同成本复制。

⑦ 批判裁决 · 全局读数

根本权衡不会消失:无本体(UMI 类)便宜($25–$400)、快(~111 演示/时,~3× 遥操作)、可众包 —— 但带本体差距(手持夹爪 ≠ 部署机器人的臂/手/底盘)且动作标签是恢复的非实测的(SLAM/VIO 6–24mm 误差、漂移、失败进训练数据)。遥操作(ALOHA/DROID/智元)给精确实测关节动作,但 ~$22k/站 + 一个人 + ~35 演示/时的疲劳天花板。

2026 各自在赢什么:无本体(A)在赢预训练规模战(Generalist 50万+时、Figure Go-Big、EgoScale 对数线性律);遥操作/真机(B/C)在赢最后一公里(接触密集/力敏感仍需干净标签,遥操作纠错 RL(π*0.6/RECAP)是当前最好的可靠性杠杆)。全行业已收敛到"便宜预训练 + 干净微调"的混合,没有严肃实验室押单一族。

趋势:A 族设备数在爆发(UMI→FastUMI→DexUMI→ActiveUMI→Mobile-UMI→Dobb·E→DexCap→手套),每个变体啃一个具体短板 —— 但每加一个能力(灵巧手/移动底盘)就加宽它必须再补的本体差距,所以硬问题(力、精确运动学、人步态→轮式)反复出现。遥操作没被取代,是被降级到标签必须精确的最后一公里。

ODM / 硬件供应商读数:数采装置有真实硬件供应机会,但是低毛利、走量的利基,不是护城河——夹爪/相机支架/动捕/VR 外设/触觉手套都是标准件组装,竞争者是任何能照 UMI 开源 BOM 拼装的厂。真正的价值在数据本身与标注管线,不在装置。
