## 1. 各家方案对比 → 设计启示 (survey → design principles)

从 10 个系统设计(UMI / FastUMI / GELLO / ALOHA 2 / DexUMI / EgoMimic / Mobile-UMI / DROID / AgiBot 采集体系 / Aria)与三族权衡矩阵(A 可穿戴 / B 主从遥操 / C 在机采集)中蒸馏出的硬设计原则,每条注明它在本设计中**强制**了什么:

- **单时钟公理(UMI GPMF 教训)。** UMI 把视频+IMU+音频锁进 GoPro 单硬件时钟,消灭了整类时间戳融合 bug,ATE 做到 6.1mm/3.5°。1 m/s 手速下 1ms 时钟误差 = 1mm 标签误差——**同步精度就是标签精度**。→ 强制:每台设备内所有模态共享一个晶振;Gen-2 起升级为 B 案的硬件公理「**任何一路数据不得携带软件时间戳**」,由锁步 Cortex-R 硬实时岛统一打戳。
- **Measured(实测)> recovered(恢复得出),但只在需要处付钱。** GELLO 12-bit 编码器(0.088°)标签误差为零;SLAM 恢复式标签地板 6–24mm(UMI 6.1mm,FastUMI/T265 10.5–24.1mm)。但 6mm 对多样性主导的预训练够用,对 0.5–2mm 间隙插装是纯噪声。→ 强制:精度按任务价值分层采购(C 案),同时用 A 案的逐误差项工程把 recovered 逼近 measured;GELLO 套件($363/臂)永久保留为**野外计量基准仪**,审计每一代 recovered 标签。
- **Inpainting(图像修补)即自认失败。** DexUMI 需要机械外骨骼重映射 + SAM2/ProPainter 视觉擦除双管齐下——"事后修补域差距"路线的成本自白。→ 强制:采集端与部署端共用同一颗传感头(B 案),视觉域差距在**构造层面**归零,不在数据管线里近似消除。
- **疲劳天花板不可谈判。** 遥操作 ~35 demos/h(每小时演示次数)是人体硬顶(iDP3:"tiring… hard to scale up"[令人疲劳……难以规模化]),$118–136/h(美国价:~$136/h 2025Q4 → ~$118/h 2026-03)。可穿戴 111 demos/h(UMI 实测)且可众包($3–5/h 中国人力)。→ 强制:走量任务永远归 W 系;L 系(主从/外骨骼)只承担测量级标签与计量职能,永不承担规模。
- **力/触觉是视频伪造不了的唯一信号。** 三族矩阵中 Family A 的结构性盲区恰是接触密集任务最缺的监督。→ 强制:力通道从 **Gen-1 第一天**上人手(A 案 ~$21 应变梁+ADC 方案,数据科学评委钦点嫁接),不等 Gen-2;触觉逐代加密(2 垫 → 100+ taxel[触觉像素点] → 1,000 taxel/指)。
- **T265 之死 = 供应链纪律。** Intel 停产 T265 直接击穿 FastUMI 一代设备。→ 强制:每个关键单源件(SoC、事件相机、F/T[力/力矩传感器]、GoPro、DYNAMIXEL)带命名第二源或自研退路,断供后 90 天内替代件必须过台架(流程化,不是口号);计算模组双源(D-Robotics + Qualcomm/Quectel 并行认证)。
- **QC(质量控制)在采集现场,不在数据中心。** 万台级机队日产 PB 级,坏数据的存储/清洗成本超过采集成本;清华 ICLR 2025 幂律说明被浪费的不是小时是**环境多样性**。→ 强制:漂移告警、自动重采、标定健康度在设备端实时闭环;QC 云同时是多样性调度市场(饱和环境赏金自动衰减)。
- **精度主张必须带日期化验收门。** 三份设计都喊 2mm,只有 C 给了"月 9 台架门禁 + 失败即转 measured 路线"的预案。→ 强制:每次精度跳变附 pass/fail 日期与回退路径(工程评委钦点)。
- **规模是唯一被证明的杠杆,且未饱和。** GEN-0 幂律(270k h 未饱和)、EgoScale 对数线性(1k→20k h 灵巧度翻倍)、清华环境多样性幂律——三条独立证据链。→ 强制:scaling ladder(扩展阶梯)以小时×多样性为主轴,精度爬升服务于它而非替代它。
- **修正数据从第一天积累。** π*0.6/RECAP 证明部署期人类干预+结果标签是 RL(强化学习)微调燃料;每个 2026 前沿配方都以此收尾。→ 强制:R 系在机改装盒(旁路侦听总线 + 干预按钮)进 Gen-1,不等 Gen-3(战略评委钦点:月 3 而非月 18 开始 DAgger 飞轮)。
- **标签保真度可以定价。** 每 episode(演示片段)附质量证书(ATE[绝对轨迹误差]上界、同步残差、力标定日期)→ 数据按认证精度分层出售,是对遥操供应商的直接定价攻击(A 案,三位评委两位钦点)。
- **保真度不需要一步到位,只需单调爬升 ≥ 大脑成熟速度。** S2 今天吃得下 6mm,S1 明天才要 2mm,接触精通后天才要千点触觉(A 案哲学,数据科学评委认定为对 scaling 证据"最正确的解读")。→ 强制:三代规格阶梯与大脑分层成熟节奏对齐,不为今天用不到的精度付 BOM(物料清单成本)。

## 2. 需求推导:大脑 scaling 决定数据规格 (brain-scaling-derived requirements)

**推理链:大脑每层的训练输入 → 标签规格 → 传感器与时钟 → 硅片。** 分层大脑事实基准见 [机上大脑](on-device-brain.md)。

**S2(慎思层,7–10 Hz,VLM[视觉语言模型])。** Helix S2 用 500+ h 遥操微调 7B(70 亿参数)VLM;EgoScale 用 20,854 h 第一视角人类视频(NVIDIA 公司口径);语义标签可事后 VLM hindsight(事后回溯)补打。推导:标签率 ≤10 Hz、位姿 cm 级即可容忍、同步 10ms 级够用;真正约束是**小时数 × 环境/物体多样性 × 语言监督**(清华幂律:环境多样性主导,单环境 demo 数饱和)。必备模态:宽 FOV(视场角)RGB(≥30 fps)、同步音频(操作者旁白 → hindsight 标注原料)、注视点(EgoMimic:90 min 眼镜数据 → 最高 +400% 协同训练增益,意图标签近零边际成本)。**结论:S2 = 量与多样性问题,由 W 系走量满足,不驱动硬件成本。注视提前到 Gen-2(B 案时点,数据科学评委钦点)。**

**S1(视觉运动层,50–200 Hz)——整个 rig(采集装置)设计的真正驱动者。** π0 动作专家 ~50 Hz / H=50 chunk(区块);GR00T N1 120 Hz;Helix S1 200 Hz / 35-DoF(自由度)。现有语料标在 15–50 Hz(DROID 15 Hz、ALOHA 2 50 Hz)——今天的 200 Hz 策略是在 50 Hz 标签上插值训练的,这是已知天花板。推导链:

- **标签率:** 训练 N Hz 策略需标签 ≥N Hz,否则插值伪影直接进梯度。Gen-1 ≥60 Hz(覆盖 π0 级,留余量);Gen-2 ≥200 Hz(对齐 Helix S1,消灭插值);Gen-3 500 Hz–1 kHz(为 S0/S1 边界高频段留余量)——Gen-2/3 数字均为 (设计目标)。
- **位置/姿态精度:** 标签噪声必须低于任务公差,否则回归目标本身是噪声。精密插入公差 0.5–2mm → 标签需 ≤1–2mm。Gen-1:recovered 6mm / measured ~1mm(GELLO 路径);Gen-2:recovered ≤2mm / 1°;Gen-3:全线 1mm / 0.5°,相对 ATE sub-1mm(亚毫米级) (设计目标)。**取 C 的保守 Gen-3 目标而非 B 的 0.5mm@1kHz——数据科学评委判定后者超出任何任务公差推导的需要,over-spec(过度规格)不付钱。**
- **力:** 人手作业 0.5–30N 指尖(Inspire/OmniHand 量程共识),插入/装配力控窗口 ~0.1–1N,冲击瞬态频谱到 ~200 Hz → 奈奎斯特要求 ≥500 Hz 采样。阶梯(Gen-2/3 为 设计目标):Gen-1 单轴抓握力 0.5N 系统级 @ 1 kSPS(**原生,非选配**);Gen-2 六轴 F/T 0.1N @ 1 kHz;Gen-3 0.01N / 0–50N (设计目标)(SharpaWave 0.005N 证明工程可行)。
- **触觉密度:** 滑移检测需 ~10² taxel/指 @ ≥200 Hz(LinkerBot 72-cell @ 200 FPS 是市售底线);纹理/剪切场需 VBT(视觉基触觉)级。阶梯(Gen-2/3 为 设计目标):Gen-1 GelSight Mini 定性($499/只);Gen-2 100+ taxel @ 500 Hz;Gen-3 1,000 taxel/指尖 @ 1 kHz (设计目标)。Digit 360 的 8.3M taxel 证明密度上限远未触及,但 64–1,000 taxel/指才是训练有效区——过密无法整手化。
- **同步:** 200 Hz 融合 → 帧周期 5ms → 跨模态对齐 ≤0.5ms;1 kHz 力/触觉瞬态 → ≤100µs(硬件触发);Gen-3 片上时间戳 ≤1–10µs,多设备 gPTP (IEEE 802.1AS) <1µs——Gen-2/3 数字均为 (设计目标)。软件时间戳自 Gen-2 起在硬件架构层出局。

**S0(反射层,500 Hz–1 kHz)+ RL 修正。** 诚实推导:S0 不直接吃人类演示——Helix 的 1 kHz 反射网训练于 1,000+ h 重定向人体运动 × 200,000 并行仿真环境。采集设备对 S0 的正确贡献是三条间接通道:(a) **全身运动重定向语料**(Gen-3 W 系加稀疏体感 IMU,喂仿真管线);(b) **接触系统辨识 ground truth(标准答案)**——真实力/振动瞬态是仿真接触模型的校准数据,即 A 案的"力数据卖两次":同一份 Gen-2 力-振动数据,策略团队与 sim(仿真)团队是两个买家;(c) **部署期干预/修正标签**(R 系:旁路侦听 500 Hz–1 kHz 本体感知 = 零成本 measured 金标签,干预按钮打 RECAP 式 correction flag(修正标记) + 成功/失败/损坏结果标签)。**Gen-3 混合体向 R 系倾斜的根本原因:届时大脑边际瓶颈已从"会做"迁到"做得可靠",可靠性数据只能来自零 gap(零差距)真机部署。**

**跨代格式底线:** LeRobot v2/v3 dataset shard(数据集分片)(AV1 视频 + parquet 动作/力/触觉流,单时钟)为主存,RLDS 一键导出,零转换税进 π0/GR00T 微调管线;每 episode 附 QC manifest(标定健康、ATE 代理、同步残差、模态清单、操作员 ID)——Gen-2 起升级为**可出售的保真度 SLA(服务级别协议)证书**。

### R1 在机改装盒(修正数据种子,提前两代启动飞轮)

## 4. Gen-2「自研模组」(months 6–15)

### 公共传感头板卡(三 SKU 共用,脊柱硬件化 + NRE(不可重复工程费用)摊薄)

- 2× onsemi **AR0234 全局快门** 2.3MP + 双 155° 鱼眼(10cm 基线立体)+ ICM-42688 **2 kHz 硬件触发**(取原生 ODR 档;该器件无 1.6 kHz 档)+ STM32H7 时间戳协处理器:所有传感器共享一条硬件触发线,快门中点打戳,**跨模态对齐 <10µs**(工程评委钦点的 C 案共用板卡方案)。
- **计算双源(B 案嫁接,T265 教训流程化):** 主选 D-Robotics **RDK S100**(¥2,799/~$390,80 TOPS BPU(带宽处理单元) + 4× Cortex-R52+ 锁步核)——R52+ 锁步核作**硬实时时间戳岛**,执行"任何一路数据不得携带软件时间戳"公理,BPU 跑在线立体 VIO(视觉惯性里程计) + 蒸馏 SAM2(人手分割/隐私模糊端侧完成);并行认证 Quectel SP895BD-AP(Dragonwing Q-8750,80 TOPS NPU,2026-01 发布)为第二源,任一断供 90 天切换。

### 精度跳变 6mm → 2mm/1° (设计目标) 的确切机制,附验收门

1. **全局快门立体 VIO** 消灭卷帘畸变(1–2 m/s 手速下 UMI 误差预算最大单项)与单目尺度漂移;
2. **AprilTag 锚垫星座**:SLAM 从"漂移问题"变"有界 PnP(透视 n 点)问题",误差不随轨迹长度增长,掠过即得 ~1mm 绝对修正;
3. **2 kHz 硬触发 IMU** 消灭振动混叠 + 同步误差项从 0.5–1mm 压到 0.1mm;
4. **在线 VIO 只做 QC,真标签由离线固定滞后平滑/光束法平差产出**(离线对在线典型增益 2–3×)。
**月 9 台架门禁(C 案,工程评委钦点):** 台架 ATE ≤2mm 且**野外中位 ≤4mm** 通过;未过 → 预案自动生效:L 系(measured 外骨骼/编码器)扩产替位,W 系退守纯预训练定位重新定价。**前置门:精度价值消融实验(A 案风险 3,数据科学评委称"三份文档中最重要的单个实验")——同一数据、仅标签 ATE 不同(6mm vs 2mm),测下游策略成功率差;若增益 <5 pts,Gen-2 精度 capex(资本开支)论点提前证伪,资源转向多样性与力/触觉。**

## 7. 终局 (the endgame of Physical AI)

**2028(可证伪):品类合并之年。** "数据采集设备"与"机器人传感头"合并为一个品类:判据 (a) 新发布商用机械臂/人形 SKU 中 ≥30% 采用第三方标准化传感头模组(2026 年为 0%);(b) 本引擎单时钟 LeRobot 兼容格式年产 ≥100 万 h;(c) 至少一个前沿 VLA(视觉语言动作模型)训练集 ≥50% 来自采集-部署同构数据,并公开消融证明同构 co-training 增益 > 异构混合(EgoMimic +400% 是方向性先例)。同年,可穿戴/ego 数据成为预训练默认来源——行业前五 VLA 中 ≥3 家预训练语料以 wearable/ego 为主,遥操退守 <5% 数据量的"纠错微调"通道(π*0.6/RECAP 模式)。**若届时遥操仍占主流模型训练小时 >30%,或部署机器人年产数据已超所有专用采集设备之和,本节时间表证伪并触发第 8 节预案。**

**2030(可证伪):倒置之年。** 数据采集完成"倒置"——机器人为机器人采集:部署舰队回传占新增数据 >70%(2026 年 AgiBot 100 台 G1 专职采集是反面形态);头部模型新增训练数据中人类演示占比 <10%;每小时验收数据现货价 <$0.5–1(2026-03 美国遥操 $118/h 的 1/100–1/200,沿 2024 $340/h → 2026 $118/h 曲线外推 + token 化上行)。专职"采集员"职业萎缩,人类角色收缩为**干预标注与偏好判断**;W/L 系转型"分布尾部补采仪器"(新场景/新物体/新接触模式定向众包),按稀缺度而非小时计价。C1 SoC 的主要出货形态是**机器人身上的 R 系记录/传感芯片**,不是手持设备。

**终态:采集行业消失,脊柱活下来。** 采集功能溶解进消费级 AR 眼镜与工作手套(Aria 1,000+ 台/300 实验室是这条曲线的起点),人类日常生活成为被动语料;硬件全部死掉或并入机器人本体。穿越终局的资产是三层(B 案的 Quectel–Qualcomm–ARM 剧本 + C 案的清算所):**传感头模组**(Quectel 式量价)、**C1 SoC 授权**(高通式,进机器人 OEM 的 BOM——触觉 AFE(模拟前端)与 VIO 前端直接就是本体感知器件,SharpaWave 进 NVIDIA/Unitree 参考设计是先例)、**时钟/格式/QC 认证云**(ARM 式生态税:任何机器人、任何传感器进,带出处、带标定健康、带精度证书的训练 token 出——物理 AI 的数据清算所)。机器人能力永远以"与人类演示的差距"计量,而我们拥有测量该差距的尺子(human-baseline 认证方)。类比:相机公司死于手机,但 MIPI/ISP 标准与图像管线活着。**终局判据(可证伪):episode manifest/精度证书格式 2029 年前被 ≥3 家非客户机构原生采用;C1 硅 2030 年前进入 ≥1 家 top-10 本体厂 BOM。两者皆空 → 终局论点破产,公司价值收敛于存量数据资产。**

## 8. 风险与判据 (risks + falsifiers)

1. **2mm 精度跳变在野外跳票。** 反光/无纹理/快动(FastUMI 最差 24.1mm 的任务依赖性警告)+ 众包锚垫合规率。**判据:月 9 台架门禁 ATE >3mm,或月 12 野外抽检 >20% 会话 ATE >4mm / 锚垫合规率 <70% → 强制转 measured 路线(L 系外骨骼/编码器扩产),W 系退守纯预训练定价。** 前置保险:精度价值消融(同数据、仅 6mm vs 2mm 标签差)若下游增益 <5 pts,capex 提前止损。
2. **具身/动力学差距吃掉规模红利。** 传感器同构 ≠ 域差距归零(手持夹爪质量分布 ≠ 机械臂);人手阻抗 ≠ 机器人手,力剖面可能误导策略(DexUMI 双管齐下已是难缠暗示)。**判据:每 5k h 固定基准 co-training 消融;同构头数据增益 ≤ 异构 OXE 混合 → B 论题失效,退守高质量 rig 厂商;力条件化训练不优于无力基线 → 力模态降级为 sim 校准专用。**
3. **仿真吃掉 S1 / 纯视频削弱标签溢价。** Helix S0 已纯仿真;EgoScale 无动作标签视频拿到对数线性增益。**判据:2027 年前出现 (a) <10% 真实数据在接触密集基准达纯真实组 95% 性能,或 (b) video-only 预训练 + <100 h 动作标签追平 10k h 级动作语料 → 产品重心转向部署态修正数据(DAgger 飞轮打不掉)与力/触觉(视频永远伪造不了)。**
4. **部署数据飞轮提前到达,Gen-2/3 资本开支搁浅。** **判据:季度跟踪头部机队公开小时数;任一 Family C 玩家 >5 万 h/年 → Gen-3 提前砍 W3 眼镜,全力转 R 系/C1 SoC——这正是组合架构对单设备对手的胜负手:混合体可以转向,单一设备只能陪葬。**
5. **供应链单点(T265 之鉴)。** GoPro GPMF(W1 命脉)、DYNAMIXEL(L 系)、RDK S100(Gen-2 计算)、Prophesee(事件相机)、C1 NRE($15–25M)。**对策已内置为流程:每关键件命名第二源(计算双源已并行认证;Gen-2 自研传感头本身就是 GoPro 退出计划;Dynamixel 国产总线舵机月 6 过台架);断供 → 90 天内替代件过台架;C1 每功能块保留 COTS 降级;无事件相机架构仍须达 2.5mm。**
6. **触觉不迁移 + 三 SKU 执行超载。** 跨 taxel 拓扑迁移若持续失败,千点触觉 ROI(投资回报率)存疑;三条硬件线 + 脊柱同时推进拉爆团队(工程评委对 C 的核心担忧)。**判据:2027 年中前无公开跨触觉硬件正迁移 → 砍 e-skin 保六维 F/T(Landot 出货曲线证明真需求);执行侧:Gen-1 的 L1/R1 全 COTS 零自研、Gen-2 三 SKU 共用一块传感头板卡,把并行 NRE 压到一份。**

## 9. 评委团意见摘要 (judge-panel summary)

- **三镜头三个赢家,总分定骨架:** 工程评委判 A 胜(51/50/39)——最佳误差预算工程学,每个同步数字都有物理机制;数据科学评委判 C 胜(51/42/41)——"数据混合体是一阶设计变量"正是前沿语料的真实组装方式,且唯有 C 从第一天供 RL 修正数据;战略评委判 B 胜(50/39/33)——"采集=部署"是唯一终局即论题、不需转身的架构,三层终态(模组/SoC/生态税)是完美的模组厂剧本。**总分 C 140 > B 131 > A 125,故骨架取 C 的 SPINE 组合(W/L/R + 脊柱),再以嫁接修其三处硬伤。**
- **修 C 的时钟松弛(工程+数据科学一致要求):** Gen-1 跨时钟域用 A 的 GoPro Media Mod 声学啁啾(<0.5ms)替换 C 的 ≤5ms 仪式;Gen-2 起采 B 的锁步 R52 时间戳岛 + "无软件时间戳"公理 + gPTP <1µs。
- **修 C 的力觉迟到(数据科学钦点):** A 的 ~$21 应变梁+24-bit ADC 原生抓握力通道进 Gen-1 W 系,接触信号从第一天进预训练语料;注视按 B 的时点提前到 Gen-2。
- **修 C 的 GTM(市场进入策略)缺口(战略钦点):** 嫁接 B 的传感头同构(采集/部署变体 BOM 差 <5%、同产线)与三层终态;嫁接 A 的 per-episode 保真度 SLA 证书与 sub-$500 ego SKU(对 $371 UMI 裸 BOM 的走量应答)。
- **修 A/B 的芯片幻想(工程钦点):** Gen-3 SoC 采 C 的封线纪律——明确不上片清单、40 TOPS@8W/~5 TOPS/W 包络、0.5W 固定功能 VIO 的占空比能效论证;否决 B 把部署策略推理塞进采集 die;NRE 修正为 $15–25M(7nm 级)。
- **修 C 的算术与执行(工程钦点):** W2 BOM 从 $650 重定价为 $800–1,400(装得下 SoC+F/T);三 SKU 共用一块传感头板卡摊薄 NRE;每个精度主张挂日期化验收门(月 9)+ 预置回退路径;GELLO 套件终身保留为野外计量基准仪。
- **两个门禁实验先于两笔最大 capex:** 精度价值消融(6mm vs 2mm,来自 A)先于 Gen-2 精度投资;同构 co-training 消融(来自 B)先于 RM 部署变体投资——评委团一致:**先证伪,后流片。**
